# Davatar

> Group portrait of contributors.

## Usage

```url
https://davatar.kecrily.me/<owner>/<repo>
```

## Self Host

1. [Generate new GitHub Token](https://github.com/settings/tokens)
2. Create `.env` file

  ```properties
  GITHUB=<Token>
  ```
